\section{Xamarin.Android}
Ähnlich wie bei Xamarin.iOS bietet Xamarin.Android die Möglichkeit native Android Apps mit Xamarin
zu erstellen. Xamarin benutzt den sogenannten Just-In-Time (JIT) Compiler um eine raffinierte
Optimierung der App Performance zur Laufzeit zu ermöglichen. D.h. eine Xamarin.Android App ist eine
native Android APK. Xamarin bringt 100\% von Google´s Android API in C\# mit. Mit dem bereits
erwähnten automatischen Binding-Generator kann man Java Code, Frameworks und benutzerdefinierte
Controls in einer Xamarin App benutzen. 
\section{NuGet und Xamarin Component Store}
Xamarin bietet die Möglichkeit Komponente zu jeder App direkt aus der IDE hinzuzufügen.
Somit kann man diverse Controls, Web Service APIs und vieles mehr benutzen. Es können auch
populäre Backends wie Microsoft Azure, Parse, Salesforce, SAP usw. in die APP integriert werden,
sowie auch Security Features wie Authentifizierung und Verschlüsselung.
